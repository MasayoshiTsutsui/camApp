# Face Recognition Programs for Mac
Face recogntion programs using the builtin camera on Mac.

# Requirement

for c++
```
clang==13.1.6
cmake==3.25.2
```

for python
```
python==3.8.13
```

# Installation

```
brew install opencv
pip install python/requirements.txt # if using python codes
```

# Usage
for c++
```
$ cd c++/build
$ cmake .. .
$ make
$ ./Main
```

for python
```
$ cd python
$ python capture.py
```
